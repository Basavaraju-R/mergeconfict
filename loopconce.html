Introduction to Loops

Loops are one of the fundamental concepts in programming. They allow you to execute a block of code repeatedly, which makes your programs more efficient and powerful.

Why Use Loops?

Imagine you need to print numbers from 1 to 100. Without loops, you'd have to write 100 print statements. With loops, you can do this with just a few lines of code!

Types of Loops

In most programming languages, you'll find these main types of loops:

  • While loops - Repeat while a condition is true
  • For loops - Repeat for a specific number of times
  • Do-while loops - Similar to while loops but execute at least once
// Example: Printing numbers with a loop
for (let i = 1; i <= 5; i++) {
  console.log(i);
}

While Loops

A while loop repeats a block of code as long as a specified condition is true.

// Syntax of a while loop
while (condition) {
  // code to be executed
}

Example: Countdown

let count = 5;
while (count > 0) {
  console.log(count);
  count--;
}
console.log("Liftoff!");

For Loops

For loops are used when you know how many times you want to execute a block of code.

// Syntax of a for loop
for (initialization; condition; increment) {
  // code to be executed
}

Example: Multiplication Table

const number = 3;
for (let i = 1; i <= 10; i++) {
  const result = number * i;
  console.log(`${number} × ${i} = ${result}`);
}

Do-While Loops

A do-while loop is similar to a while loop, but it always executes the code block at least once before checking the condition.

// Syntax of a do-while loop
do {
  // code to be executed
} while (condition);

Example: User Input Validation

let userInput;
do {
  userInput = prompt("Enter a number greater than 10:");
} while (userInput <= 10);
console.log("Thank you!");
